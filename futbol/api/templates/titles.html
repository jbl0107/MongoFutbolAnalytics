Información sobre este Documento

En la documentación de swagger (situada en la url 127.0.0.1/8000/docs/) no pueden realizarse operaciones POST. Si desea probar al completo la API, pongase en contacto con nosotros a través del correo que se muestra en la documentación comentada anteriormente (en la sección "Contact the developer") para recibir una invitación al workspace de Postman donde todas las peticiones pueden ser probadas. Para probar la peticion POST de este Documento en Postman, deberá seleccionar la opción raw-JSON de Body.

Cuando en una URL se indica que debe especificar el nombre de una liga, tipo de gol o competición, y la que desea consultar tiene espacios, debe sustituir estos por "-". Un ejemplo: si desea buscar información sobre la competición "Premier League", en la URL deberá aparecer Premier-League.

• Los siguientes conjuntos de atributos son únicos: (name, year), (name, team, year).

• Rango de valores para "year": [1888, 2023]

• El atributo "name" solo puede tomar estos valores:
