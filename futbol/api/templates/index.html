¿Qué puede encontrar aquí?

En nuestra api podrá encontrar estadísticas detalladas sobre equipos, jugadores y partidos de fútbol. Utilizamos una base de datos MongoDB para almacenar y consultar los datos de manera eficiente.

¿Por qué MongoDB?

Decidimos utilizar MongoDB para nuestro proyecto porque es una base de datos NoSQL flexible y escalable que nos permite almacenar y consultar grandes cantidades de datos de manera eficiente.
