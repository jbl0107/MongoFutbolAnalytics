Información sobre este Documento

En la documentación de swagger (situada en la url 127.0.0.1/8000/docs/) no pueden realizarse operaciones POST. Si desea probar al completo la API, pongase en contacto con nosotros a través del correo que se muestra en la documentación comentada anteriormente (en la sección "Contact the developer") para recibir una invitación al workspace de Postman donde todas las peticiones pueden ser probadas. Para probar la peticion POST de este Documento en Postman, deberá seleccionar la opción raw-JSON de Body.

Cuando en una URL se indica que debe especificar el nombre de una liga, tipo de gol o competición, y la que desea consultar tiene espacios, debe sustituir estos por "-". Un ejemplo: si desea buscar información sobre la competición "Premier League", en la URL deberá aparecer Premier-League.

Objeto JSON

Este es un ejemplo del JSON de un objeto del Documento Goal. Todos los atributos son obligatorios excepto "assistant".
• El siguiente conjunto de atributos es único: (player, team, year, competition, minute).
• Valor máximo para "minute": 120.
• Rango de valores para "year": [1888, 2023]
• El atributo "team" hace referencia a un objeto del Documento Team.
• Los atributos "player" y "assistant" hacen referencia a un objeto del Documento Player.
• El atributo "local" hace referencia a si el equipo que recibe el gol lo hace siendo local (valor a True) o visitante (valor a False).
• El atributo "competition" solo puede tomar estos valores:

  • Champions League
  • Europa League
  • LaLiga
  • Copa del rey
  • Premier League
  • Community shield
  • Ligue 1
  • Coupe de la Ligue
  • Serie A
  • Coppa Italia
  • Bundesliga
  • DFB-Pokal

• El atributo "type" solo puede tomar estos valores:

  • De cabeza
  • Penalti
  • Tiro libre
  • Corner
  • Chilena
  • Pierna izquierda
  • Pierna derecha
  • Dentro del área
  • Fuera del áera
